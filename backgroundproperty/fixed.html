background attachment - fixed

The background-attachment property sets whether
a background image scrolls with the rest of the page, or is fixed.

The background-attachment is the Property used to control the image how we want to use it.




Property Values


scroll : The background image will scroll with the page. This is default

fixed :The background image will not scroll with the page

local :The background image will scroll with the element's contents

initial :Sets this property to its default value.he initial keyword is used to set a CSS property to its default value.
The initial keyword can be used for any CSS property, and on any HTML element.

inherit :Inherits this property from its parent element.The inherit keyword specifies that a property should inherit its value from its parent element.
The inherit keyword can be used for any CSS property, and on any HTML element.
